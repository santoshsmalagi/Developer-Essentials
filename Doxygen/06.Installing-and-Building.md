## Installing and Building

#### CentOS 7

Update to the latest version of gcc

```Shell
sudo yum -y update                                                                           # update the system
sudo yum -y install bzip2 wget gcc gcc-c++ gmp-devel mpfr-devel libmpc-devel make            # install dependencies (if not available already)
wget http://mirror.rit.edu/gnu/gcc/gcc-9.3.0/gcc-9.3.0.tar.gz                                # download gcc source from nearest mirror
tar -xvzf gcc-9.3.0.tar.gz                                                                   # untar
cd gcc-9.3.0
mkdir build                                                                                  # create a build dir to avoid messing up source code
cd build
../configure --enable-languages=c,c++ --disable-multilib
sudo make
sudo make install                                                                            # default intall at /usr/local
export PATH=/usr/loca/bin:${PATH}                                                            # add to path
export LD_LIBRARY_PATH=/usr/local/lib64:${LD_LIBRARY_PATH}                                   # add to linker search dir path
```
